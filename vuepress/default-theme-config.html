# 侧边栏

想要使 侧边栏（Sidebar）生效，需要配置 themeConfig.sidebar，基本的配置，需要一个包含了多个链接的数组：

// .vuepress/config.js
module.exports = {
  themeConfig: {
    sidebar: [
      '/',
      '/about',
      ['/mark_md', 'Markdown 语法简易记']
    ]
  }
}

# 侧边栏分组

你可以通过使用对象 sidebar:[{},{}] 来将侧边栏划分成多个组：

// .vuepress/config.js
module.exports = {
  themeConfig: {
    sidebar: [
      // 第一组
      {
        title: 'C++技术文档',
        path: '',
        collapsable: true,
        sidebarDepth: 2,
        children: [
          '/cpp/',
          '/cpp/basic',
          '/cpp/example',
        ]
      },
      // 第二组
      {
        title: 'Vuepress配置',
        children: [
          ['/vuepress/', '介绍'],
          '/vuepress/default-theme-config'
        ]
      }
    ]
  }
}

# 多个侧边栏

如果你想为不同的页面组来显示不同的侧边栏，遵循以下的侧边栏配置：

// .vuepress/config.js
module.exports = {
  themeConfig: {
    sidebar: {
      // cpp页面下才显示该侧边栏
      '/cpp/': [
        '',     /* /cpp/ */
        'basic',  /* /cpp/basic.html */
        'example'   /* /cpp/example.html */
      ],
      // vuepress页面下显示该侧边栏
      '/vuepress/': [
        '',      /* /vuepress/ */
        'theme-default-config', /* /vuepress/theme-default-config.html */
      ],
      // fallback
      '/': [
        '',        /* / */
        'mark_md', /* /mark_md.html */
        'about'    /* /about.html */
      ]
    }
  }
}

# 嵌套侧边栏

这里，结合了多个侧边栏和嵌套侧边栏的设置。假设一个需求：你想在 /cpp/ 路径下显示它该有的分组侧边栏，在 /vuepress/ 路径下又显示出和前者不一样的分组侧边栏，同时又希望在侧边栏中 /vuepress/ 的显示标题（介绍）和 /vuepress/README.md 里的标题（Vuepress指南）不一致。那么可以综合上述，进行如下配置：

// .vuepress/config.js
module.exports = {
  themeConfig: {
    sidebar: {
      // cpp页面下才显示该侧边栏
      '/cpp/': [
        // 分组1
        {
          title: 'C++技术文档',
          path: '',
          collapsable: true,
          sidebarDepth: 2,
          children: [
            '/cpp/',
            '/cpp/basic',
            '/cpp/advance',
            '/cpp/file',
            '/cpp/generic',
            '/cpp/stl',
            '/cpp/stl_common_use',
            '/cpp/stl_object',
            '/cpp/example',
          ]
        },
        // 分组2
        {
          title: 'C++问题解决方案',
          children: [
            '/cpp/question/q1',
            '/cpp/question/q2'
          ]
        }
      ],
      // vuepress页面下显示该侧边栏
      '/vuepress/': [{
        title: 'Vuepress配置',
        children: [
          ['/vuepress/', '介绍'],   // 另起标题
          '/vuepress/default-theme-config'
        ]
      }]
    }
  }
}
