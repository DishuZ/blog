# 零基础编程推荐

前言

在众编程语言中，我围绕着"开发一个应用"的分工来分类：

  1. 前端开发语言（即 网站、小程序、手机app的应用布局）：html + css + js 、wxml + wxss + js 、 Vue 等
  2. 后端开发语言（即 应用的业务逻辑、完成前端与数据库的连接）：PHP、C#、JAVA、Python等

我推荐入门学习Python，它有许多优点，列举几个比较吸引人的：

  1. 爬虫。比如：收集酒店网站上的所有酒店相关数据
  2. 数据分析（和Excel差不多）。比如：利用酒店数据，分析酒店的价格与酒店所在地区是否有相关性，并且可视化数据
  3. 搭建网站。利用Python写出来的框架 Django 可以用来搭建Web网站
  4. 机器学习

下面我讲针对Python语言进行相关推荐，如发现资源不适合你，可以自己去找找看。

注意

入门IT，一定要记住，不需要一下子就把全部知识搞明白，只需要先掌握最主要的就行，其他的可以在忘记/需要的时候再Search浏览器。

# Python

建议

  1. 视频、书籍、博文、学习网站，随你搭配。最终都要落在动手实践上！

  2. Python有2.x和3.x两种大版本，在这里只推荐学习Python3.x系列，如果你找到的教程是Python2.x的，建议直接丢弃。

  3. "博文"和"学习网站"部分只是扩展，而不是必须

# 视频

  • [b站]8天入门Python (opens new window)
    • 前4天讲的是Python编程基础；
    • 第5、6天讲的是面向对象编程，按视频的顺序听，前面几节要听，后面节听不懂没关系，有个大致印象就很不错；
    • 第7天讲异常处理和模块 包引用，模块和包是连接py文件的关键；
    • 第8天讲的是网络通信，这一块在实践中相对较少用过，可听可不听，也可以后再听。

# 博文

如果你喜欢博文形式的阅读学习，可以尝试这个：Python教程-廖雪峰 (opens new window)，他写的很全面也很细致。但我个人并不首先推荐看这个入门，因为廖神写的文字对我来说还是比较晦涩难懂的。

# 学习网站

在这些网站上，你可以找到一些符合你自身的教程

  • 慕课网 (opens new window)，这个网站是我入坑IT的地方，是专注编程方面教程的网站，目前有收费课程也有免费课程，免费课程已经足够看了，有些课程是视频，有些教程是在线编程。
  • B站，总所周知，b站是个学习网站。但网站里鱼龙混杂，一般是挑最多点击的教程来看适不适合自己
  • 网易云课堂 (opens new window)，这个网站不单单是学习IT，还可以学习其他任何技能。
